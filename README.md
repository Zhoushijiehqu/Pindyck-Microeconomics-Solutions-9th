# 平狄克《微观经济学（第九版）》课后习题全解答

这个仓库是我在学习平狄克《微观经济学（第九版）》时，用 LaTeX 重新排版和整理的课后习题详细解答。

一开始在网上找答案，发现大部分都是模糊的扫描版或者排版混乱的 Word 文档，看起来非常痛苦。索性自己动手，用纯 LaTeX 结合 TikZ 绘图把题目重新做了一遍。

如果你也在备考、复习，或者对经济学图表的 LaTeX 排版感兴趣，希望这份文档能帮到你。

## ✨ 特色说明

这份文档的排版标准主打“强迫症友好”，尽量向顶刊和学术出版物的标准靠拢：

1.  **极简排版**：采用 `ctexbook` 文档类，去除了冗余装饰。
2.  **纯矢量经济学图表**：全书所有的供需曲线平移、最高限价模型、弹性分析等图表，均由 `TikZ` 原生代码绘制。图表的坐标交点由底层数学方程精确计算，放大不失真。
3.  **学术规范的三线表**：使用 `tabularx` 和 `booktabs`，表格宽度 100% 撑满页面，告别丑陋的竖线。
4.  **清晰的逻辑推导**：核心公式单独成框。解题过程拒绝机械式的罗列，注重经济学直觉，严格区分“需求量的变动”与“需求的变动”等核心概念。

## 🛠️ 关于源码

如果你想基于此文档进行修改或者学习里面的 TikZ 绘图代码，你可以克隆本仓库并在本地编译。

* **主文档**：`main.tex`
* **推荐环境**：TeX Live 2023 及以上
* **编译要求**：必须使用 `XeLaTeX` 进行编译（以支持 `ctex` 宏包和中文字体）。为了确保目录正确生成，建议连续编译两次。
